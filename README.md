HZWebViewController
===================

Background blur the webview, displays the loading progress. The interface is simple and refreshing

###Usage

Add a pod entry to your Podfile:

	pod 'HZWebViewController', '~> 0.0.1'

Install the pod(s) by running:

	pod install
